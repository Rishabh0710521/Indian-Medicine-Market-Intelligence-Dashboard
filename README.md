# Indian-Medicine-Market-Intelligence-Dashboard
Indian Medicine Market Intelligence Dashboard
# Objective--
Project Objective (GitHub ReadMe/Project Post):
To build a professional, analytics-driven Power BI dashboard for the Indian Medicine Market, enabling users to visualize, compare, and gain actionable insights into medicine distribution, pricing, manufacturer activity, and market trends using a unique, rarely-used dataset.
# 7 Bullet Points â€“ Dashboard Features/Insights
# 1 Comprehensive Market Overview:
Shows total number of medicines and actively sold products using dynamic summary KPIs.

# 2 Manufacturer Performance:
Ranks manufacturers by quantity of medicines, highlighting market leaders in a clean horizontal bar chart.

# 3 Pack Size Expansion:
Visualizes top medicine pack sizes to uncover production and purchase trends in packaging.

# 4 Price Segmentation:
Segregates medicines into price bands, revealing the most affordable and premium segments across the market.

# 5 Type Distribution:
Displays the dominance of Allopathy (or any category) with interactive donut charts for clear category split.

# 6 Market Activity Status:
Differentiates active and discontinued medicines, delivering immediate business and compliance insights.

# 7 High Interactivity & Usability:
Carefully formatted visuals, easy-to-read labels, and potential slicers for instant drill-downs by type/manufacturer/pack size.
